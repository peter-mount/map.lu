Reusing map.lu content

The following rules apply when using these layers - the full details are listed at https://map.lu/about/index.html.

map.lu is is free for reuse for non-commercial purposes as long as you stay within the various copyrights of the underlying data and your traffic is not excessive. If you do use the map tiles please let me know.

Please note that bulk downloading of tiles is prohibited.

If you want to use the map data for commercial use, or you expect your traffic to be heavy then please get in touch. The main concern here is hosting and transfer costs of the map tiles.
